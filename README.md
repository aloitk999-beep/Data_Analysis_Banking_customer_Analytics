##  مشروع تحليل بيانات عملاء بنك | Banking Customer Analytics
# أهداف المشروع:
- تحليل سلوك العملاء المالي (Financial Behavior)
- تتبع أنماط الإنفاق وطرق الدفع
- تقسيم العملاء حسب الدخل، العمر، والجنس
- تقييم القدرة على السداد ومستوى المخاطر
- اكتشاف العمليات المشبوهة (Fraud Indicators)
 # أهم المؤشرات والتحليلات:
- إجمالي المعاملات وحجم الإنفاق
- متوسط قيمة العملية ونسبة الإنفاق إلى الدخل
- Credit Score & Debt Analysis
- توزيع العملاء حسب Income Segment و Age Segment
- مقارنة Swipe / Chip / Online Transactions
- تحليل العمليات المشبوهة
- الإنفاق السنوي واتجاهاته
  # هذا المشروع ساعد في
- تحسين إدارة المخاطر
- فهم سلوك العملاء
- دعم استراتيجيات التسويق والائتمان
- تعزيز كشف الاحتيال
